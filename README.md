# Solo Miner ARM64/CPU Android

Ein professioneller Android CPU-Miner für **Solana (SOL)**, **Polkadot (POL)**, **Monero (XMR)** und **JA-Coin** mit optimierter ARM64-Unterstützung.

## ✨ Features

- 🪙 Multi-Coin Support (SOL, POL, XMR, JA)
- ⚡ ARM64 CPU-Mining Optimierung
- 📊 Real-time Monitoring & Statistics
- 🔗 Mining Pool Integration
- 💡 Solo Mining Support
- 🎨 Material Design 3 UI
- 🔋 Energieverwaltung
- 📱 Android 11+ (API 30)

## 📋 Anforderungen

- Android 11+ (API Level 30)
- ARM64 Architektur
- Mindestens 2GB RAM
- Kotlin 1.9+
- Gradle 8.0+

## 🚀 Installation

```bash
git clone https://github.com/Breitjoda/solo-miner-amr64-cpu.git
cd solo-miner-amr64-cpu
./gradlew build
./gradlew installDebug
```

## ⚙️ Konfiguration

1. App öffnen
2. Wallet-Adresse eingeben
3. Coin-Typ auswählen
4. CPU-Threads konfigurieren
5. Mining starten
